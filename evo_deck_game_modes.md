# Universal Evo Deck: Game Modes

## Overview
Six distinct games playable with the single 80-card evolutionary traits deck. Each mode emphasises different card attributes.

---

## Game 1: Evo-Rummy

**Players:** 3-7  
**Time:** 30-45 min  
**Key Attributes:** MYA (runs), System suit (sets), Clade markers (bonus sets)

### Setup
- Deal 7 cards (3-4 players) or 5 cards (5-7 players)
- Flip one card to start discard pile

### Turn
1. Draw (deck or discard)
2. Meld (optional)
3. Extend existing melds (optional)
4. Discard

### Melds
| Type | Requirement | Example |
|------|-------------|---------|
| Evolutionary Run | 3+ same system, MYA gaps ≤100 MY | Notochord→Jaws→Bony Skeleton |
| Clade Set | 3+ same clade marker | 🐁 Hair + Placenta + Neocortex |
| Convergent Set | 3+ cards with 🌿 | Eyes + Wings + Warm-blood |

### Scoring
- Run/Set cards: 10 pts each
- Tight run (≤50 MY gaps): +5 per card
- Full system in clade set: +15
- Convergent set: 15 pts per card
- Caught with cards: -3 (ancient), -5 (standard), -8 (recent)

### Going Out
Empty hand with at least one meld down.

## Game 2: Timeline

**Players:** 2-8  
**Time:** 20 min  
**Key Attributes:** MYA only

### Setup
- Each player gets 4 cards face down (don't peek at MYA)
- Place one card face up as timeline start

### Turn
1. Look at one of your cards
2. Place it where you think it belongs chronologically
3. Flip to reveal - correct placement = keep it; wrong = discard

### Winning
First to correctly place all 4 cards.

### Variant: Draft Timeline
- Players see their cards
- Take turns placing
- If wrong, card goes to opponent

## Game 3: Ancestor Auction

**Players:** 3-6  
**Time:** 25 min  
**Key Attributes:** MYA (bidding), Clade markers (reveal)

### Setup
- Each player gets 500 MY "budget"
- Dealer holds deck

### Round
1. Dealer reads trait name only (no MYA)
2. Players secretly bid MY amount
3. Closest to actual MYA without going over wins the card
4. Winner subtracts their bid from budget

### Scoring
- Card value = MYA ÷ 10
- Most points when budgets exhausted wins

### Variant: Clade Bonus
Before bidding, dealer reveals clade marker. Players with cards from that clade get +50 MY budget that round.

## Game 4: Cladogram Builder

**Players:** 2-4  
**Time:** 40 min  
**Key Attributes:** MYA, Clade markers, Prerequisites, Enables

### Setup
- Lay out empty tree template
- Deal 10 cards each
- Place "Multicellularity" as root

### Turn
1. Play a card to the tree
2. Must connect to a valid prerequisite
3. MYA must be later than parent node
4. Draw a card

### Scoring
- Correct placement: +5
- Wrong placement: -5, card returns to hand
- Complete branch (5+ connected cards): +20
- First to empty hand: +30

### Verification
Use prerequisite/enables text to validate connections.

## Game 5: Do I Have It?

**Players:** 3-8  
**Time:** 15 min  
**Key Attributes:** Clade markers, System suits

### Setup
- Prepare organism cards separately:
  - Hagfish (🐟, no jaws)
  - Shark (🐟, jaws, no bones)
  - Frog (🐟🦎, no amniotic egg)
  - Turtle (🦎, shell)
  - Pigeon (🐦)
  - Platypus (🐁, eggs, no nipples)
  - Human (🐁)
  - Octopus (🦑)

### Turn
1. One player draws secret organism
2. Others take turns playing trait cards as questions
3. Organism player answers "Yes I have it" or "No"
4. First to guess organism wins round

### Scoring
- Correct guess: 10 pts
- Organism player survives full round: 5 pts

## Game 6: Extinction Roulette

**Players:** 4-7  
**Time:** 30 min  
**Key Attributes:** MYA (survival threshold)

### Setup
- Deal 5 cards each
- Shuffle extinction event cards:
  - End Ordovician (444 MYA)
  - Late Devonian (372 MYA)
  - End Permian (252 MYA)
  - End Triassic (201 MYA)
  - K-Pg (66 MYA)

### Round
1. Flip extinction event
2. All traits with MYA < extinction date survive
3. Traits with MYA > extinction date lose 1 "life"
4. Each card has 2 lives; at 0 lives, discard

### Survival
- Last player with cards wins
- Tie: highest total surviving MYA

### Strategy
Ancient traits are safer; recent traits are powerful but vulnerable.

## Recommended Progression

**For new players:**
1. Timeline (simplest, learn cards)
2. Do I Have It? (social, learn clades)
3. Evo-Rummy (full mechanics)

**For experienced players:**
- Cladogram Builder (strategic depth)
- Extinction Roulette (push your luck)
- Ancestor Auction (memory + bidding)
