# Universal Evolutionary Traits Deck

## Overview
A single 80-card deck of animal kingdom evolutionary traits designed to support multiple game modes. Cards contain all information needed for Evo-Rummy, Timeline, Cladogram Builder, and other variants.

---

### Card Elements

| Element | Purpose | Game Use |
|---------|---------|----------|
| System Icon | Suit identifier | Rummy sets, colour matching |
| Trait Name | Primary identifier | All games |
| MYA Value | Numerical age | Rummy runs, Timeline, Auction |
| Era Code | Geological period | Thematic grouping, variants |
| Clade Symbol | Taxonomic group | Set bonuses, "Do I Have It?" |
| Prerequisites | What evolved first | Cladogram Builder |
| Enables | What this unlocked | Cladogram Builder, combos |
| Flavour Text | Fun/educational | Engagement |
| System Bar | Visual suit ID | Quick recognition |

## Balancing Notes

### MYA Distribution
- 600-400 MYA: 20 cards (ancient foundation)
- 400-200 MYA: 25 cards (vertebrate radiation)
- 200-50 MYA: 20 cards (mammal/bird evolution)
- 50-0 MYA: 11 cards (recent refinements)

### Convergent Traits (🌿)
Deliberately over-represented to enable interesting set-building:
- Eyes (camera, compound)
- Warm-bloodedness
- Viviparity
- Echolocation
- Flight structures
- Bioluminescence
- Venom

### Run Accessibility
Every system has cards spanning at least 300 MY to ensure runs are achievable.
